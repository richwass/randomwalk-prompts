# Contributing

Thanks for your interest in this prompt library.

## Current Policy

This is a curated set of prompts being actively refined through real-world usage. I'm **not accepting pull requests at this time**.

## How to Share Feedback

The best way to contribute is through [Issues](../../issues):

- **What worked**: Which prompts produced output you actually used? What made them effective?
- **What didn't**: Where did the AI give generic advice instead of pushing you? Where did you have to fight the Socratic format?
- **What's missing**: Did you hit a moment where you thought "it should have asked me about X"?
- **Model differences**: Did you notice different behavior between Claude and ChatGPT?

This feedback directly shapes how the prompts evolve.
